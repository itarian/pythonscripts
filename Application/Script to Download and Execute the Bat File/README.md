NOTE:

Parameters to be edited:

Folder_name=r"Vicky"                                       # Give the new folder name
Folder_Path=r"C:\Users\rainbow\Desktop"            # Provide Path for the New folder

File_name=r"multi.bat"                                      #Provide the Bat file name with extension
