Please use the script to kill a particular application that causes your Windows Computer hangs on. You should use the program name and its extension like given in the script as well as in the Script data section.

Note:

Please run the script as System User

 

Parameter Configuration:

                      >)  Type                 :  String

                      >)  ITSM Label     :   Application Name with the Extension (Eg. chrome.exe)

                      >)  Default Value :   chrome.exe


Run as LocalSystem User
